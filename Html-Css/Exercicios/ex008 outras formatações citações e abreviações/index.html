Com e sem a tag PRE

Sem a tag PRE.

num = int(input'Digite um número')) if num % 2 == 0 print(f'O númeor{num} é PAR') esle: print(f'0 númeor {num} é ÍMPAR) print('Fim do programa')

Com a tag PRE

        
num = int(input'Digite um número'))
if num % 2 == 0
    print(f'O númeor{num} é PAR')
esle:
    print(f'0 númeor {num} é ÍMPAR)
print('Fim do programa')
